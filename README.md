## Who We Are

**Noctyrixa** is a small dev team focused on building clean, practical software, from game frameworks to web tools. We care about developer experience, performance, and code that’s easy to extend.

### What We Do

- **Framework development** — Core systems and abstractions that other projects can build on
- **Game & server tooling** — Resources and infrastructure for FiveM and similar platforms
- **Web & backend** — APIs, dashboards, and full stack apps when the project calls for it
- **Open source** — We publish what we build so others can learn from it and contribute
